## Building APK

APK is automatically built via GitHub Actions on every push.

### Download APK:
1. Go to [Actions tab](../../actions)
2. Click latest successful build
3. Download `levelup-server-apk` artifact
4. Extract and install on Android

## Commands
- `/lw{teamcode}` - Auto start for team
- `/stop` - Stop auto start
- `/help` - Show help
